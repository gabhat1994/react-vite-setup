### How to install

- pull repo
- `yarn install`

### Setup environment variable

- run `cp .env.template .env.development`

### How to start development

- `yarn dev`

### How to run unit & integration tests

- `yarn test`

### How to run E2E tests

- `yarn run cypress:run:point-to-local`
